Equilibrium

What remains when the path is forgotten.

Equilibrium is not a single destination; it can be convergence, capture (absorbing), slow forgetting (mixing), or long-lived pseudo-equilibria (metastability).

The unifying principle is a Markov lens: the system remembers only its current state.

Convergence
Systems converge to equilibrium.
Repeated transitions reveal a stationary distribution.
Absorbing State
Not all equilibria are reversible.
Once entered, the absorbing state captures all mass.
Mixing Time
Some systems forget faster than others.
Distance to equilibrium over iterations, from different initial states.
Metastability
Stable—until it isn't.
Two basins with rare transitions create long-lived pseudo-equilibria.
