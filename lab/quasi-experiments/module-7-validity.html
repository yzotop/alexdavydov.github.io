Модуль 7. Практика валидности

TL;DR

Ни один квазиэксперимент не доказывает каузальность "по формуле". Вся сила — в проверках: placebo, falsification, sensitivity. Чем больше тестов выживает — тем сильнее аргумент.

Арсенал проверок

ТестИдеяПрименимость
Placebo treatmentПодставить фиктивный treatment (другой период/группу) — эффекта быть не должноDiD, SCM, RDD
Placebo outcomeПроверить outcome, который не должен реагировать на treatmentВсе методы
Falsification testТест на outcome с известным результатом (или отсутствием эффекта)Все методы
Pre-trend testКоэффициенты event study до treatment ≈ 0DiD
McCrary density testНет скачка плотности running variable у порогаRDD
Balance testКовариаты не имеют разрыва у порога / одинаковы после matchingRDD, Matching
Sensitivity analysisНасколько сильный unobserved confounder нужен для объясненияMatching, DiD
Bandwidth robustnessРезультат устойчив при разных окнахRDD
Permutation inferenceP-value через перестановки assignmentSCM, DiD

Логика аргументации

Квазиэксперимент — не доказательство, а аргумент. Сила аргумента определяется:

  1. Правдоподобностью дизайна (история assignment)
  2. Количеством пройденных проверок
  3. Прозрачностью ограничений
  4. Устойчивостью к альтернативным спецификациям
