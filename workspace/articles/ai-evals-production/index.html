AI-evals в продакшене на маркетплейсе

Статья в работе. Будет опубликована на этой странице после завершения. Тематика — практический подход к AI-evaluation в продакшене маркетплейса: что измерять, какие методологические ошибки подстерегают, какие инструменты помогают.

До публикации можно посмотреть три интерактивных симулятора, которые иллюстрируют ключевые проблемы:

  • LLM A/B Test Simulator — sample size для сравнения LLM-моделей с поправкой на stochasticity
  • LLM-as-Judge Position Bias — mirror correction для pairwise judgment
  • Category Matching Eval — heterogeneous accuracy + stratified evaluation
